About Me

Hi, I am a 19 year old student currently studying Bsc Computer Studies at Middlesex University Malta. This website was created as an assignment for the Web Design and Databases Module. It is based from Javascript, HTML, CSS and some basic PHP to keep the header and the footer constant.

Bloxo is an infinite sidescroller game that I created to demonstrate my knoledge of javascrpt in the web. The design of the game is based off minimalism. It features clean lines and blocks which gives the game a modern feeling. The scope of the game is to take Bloxo as further away as possible withouth hitting any obstacles.
